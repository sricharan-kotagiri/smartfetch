# SmartFetch Logo

Place the `sf.jpeg` logo file in this directory.

The logo should be:
- Format: JPEG
- Filename: `sf.jpeg`
- Recommended size: 200x200px or larger
- Should work in both light and dark themes

The logo is imported and used in:
- Navbar (45px height)
- Landing page hero (80px height)
- Login/Signup pages (60px height)
- Dashboard (45px height in navbar)
