================================================================================
                         🔌 API ENDPOINTS
================================================================================

Authentication:
   POST   /api/auth/send-otp              - Send OTP via WhatsApp
   POST   /api/auth/verify-otp            - Verify OTP & Login
   GET    /api/auth/otp-status/:phone     - Check OTP status

Shopkeepers:
   POST   /api/shopkeepers/create-shop    - Create shop
   GET    /api/shopkeepers/profile/:id    - Get shop profile

Products:
   POST   /api/products/add-product       - Add product
   GET    /api/products                   - Get all products

Orders:
   POST   /api/orders/create-order        - Create order
   GET    /api/orders/user/:user_id       - Get user orders

Health:
   GET    /health                         - Health check

   2. Backend sends OTP via Twilio WhatsApp
   3. User receives OTP on WhatsApp
